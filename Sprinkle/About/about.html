Smart water quality monitoring (SWQM) system

Sprinkle : Water Quality Checker


Introduction of the App

Water is an important element for sustaining any living beings in the world. But it becomes like a threaten when it gets polluted. From the need of drinkable water, we researched on four water parameters which are pH, electric conductivity, turbidity, temperature and established a smart water quality monitoring (SWQM) system application. We named it Sprinkle : Water Quality Checker

How does it work?

This application system takes input of four parameters through four sensors with the help of Arduino. Hence, the application applies machine learning algorithm on the collected data and predicts the output as “Drinkable” or “Not Drinkable”.
